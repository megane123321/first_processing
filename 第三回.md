# 初めてのprocessing 第三回 関数を作る
processingでは関数を自作することができる。
[図1](#図1)に示す通り、書き方はsetupなどと同じである。そのためこれからはsetupやkeyPressed等も関数と呼称することとする。

<div id="図1"></div>

```java
void setup(){
  myFunc();//自作関数の呼び出し
}

void myFunc(){//関数作成
  println("自作関数が実行された。");
}
```
図1. 関数を自作する

関数の宣言は、以下の要素から成り立つ。
* 戻り値の型
* 関数名
* 引数
* 処理

`戻り値の型 関数名(引数){処理}`
